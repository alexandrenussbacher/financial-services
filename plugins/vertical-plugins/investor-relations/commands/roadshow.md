# /roadshow — Preparação de Roadshow

Prepara o pacote completo para um roadshow com investidores institucionais: narrativa IR, briefing de reunião e material de suporte.

## Como usar

```
/roadshow [nome da empresa] [tipo: IPO | follow-on | captação privada | M&A] [investidor-alvo]
```

## O que este comando produz

1. **Equity Story** — Narrativa IR estruturada, pronta para o roadshow
2. **One-Pager** — Resumo executivo de 1 página para envio prévio ao investidor
3. **Briefing da Reunião** — Contexto do investidor, agenda, talking points
4. **FAQ de Due Diligence** — Respostas preparadas para as 10 perguntas mais prováveis

## Exemplos de uso

```
/roadshow "Grupo Exemplo" IPO "fundos de pensão brasileiros"
/roadshow "Empresa XYZ" "captação série B" "family offices SP"
/roadshow "ABC Logística" M&A "investidores estratégicos internacionais"
```
