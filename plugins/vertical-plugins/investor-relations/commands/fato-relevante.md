# /fato-relevante — Análise de Fato Relevante CVM

Interpreta fatos relevantes e comunicados ao mercado sob a ótica do investidor institucional.

## Como usar

```
/fato-relevante [cole o texto do fato relevante aqui]
```

## O que este comando produz

1. **Classificação** — Positivo / Neutro / Negativo / Ambíguo
2. **Resumo executivo** — O que aconteceu em 3 linhas
3. **Impacto na narrativa IR** — Como afeta a equity story
4. **Reação esperada do mercado** — O que analistas vão perguntar
5. **Ações de RI recomendadas** — Lista priorizada de próximos passos
6. **Checklist de compliance CVM** — Verificação dos requisitos da Instrução 358
